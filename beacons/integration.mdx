Integrating Beacons takes less than a minute. Copy the snippet from Overmind's Beacons page, paste it into your site's HTML, and your floating chat widget is live.

<Steps>
  <Step title="Open the Beacons page">
    In the Overmind top bar, click the **Beacons** button. This opens the Beacons page where you can view and copy your personal embed code.
  </Step>
  <Step title="Copy the snippet">
    Click the **Copy** button on the Beacons page. The full embed code — with all the necessary connection details already filled in — is copied to your clipboard.
  </Step>
  <Step title="Paste into your HTML">
    Open your website's HTML file and paste the snippet anywhere inside the `<body>` tag — typically just before `</body>`.
  </Step>
  <Step title="Open your page">
    Load or refresh your website. You'll see the 💬 button appear in the bottom-right corner. Click it to verify that the chat panel opens correctly.
  </Step>
</Steps>

## What the Snippet Does

The snippet you copy from Overmind is a self-contained `<script>` block. You don't need to configure anything — all the connection details are pre-filled when you copy it from the Beacons page. Here is a conceptual overview of what the snippet does:

```html
<script type="module">
  // 1. Connects to Overmind's backend to fetch the chat interface URL
  //    (connection details are pre-filled in the snippet you copy)

  // 2. Creates a floating 💬 button fixed to the bottom-right of the screen
  const btn = document.createElement("div");
  btn.textContent = "💬";
  // ... button styles applied automatically

  // 3. Creates a slide-in overlay panel to host the chat interface
  const panel = document.createElement("div");
  // ... panel styles applied automatically

  document.body.appendChild(btn);
  document.body.appendChild(panel);

  // 4. Toggles the panel open/closed on button click
  btn.onclick = () => {
    panel.style.transform =
      panel.style.transform === "scale(1)" ? "scale(0)" : "scale(1)";
  };

  // 5. Loads the Overmind chat URL into an iframe inside the panel
  async function load() {
    const { data } = await supabase.from("scripts").select("url").maybeSingle();
    if (data?.url) {
      panel.innerHTML = `<iframe src="${data.url}" style="width:100%;height:100%;border:none;"></iframe>`;
    }
  }

  load();
</script>
```

<Note>
  Never share or publish the snippet's raw connection details. The snippet is intended for your own website only — treat it like a private API key.
</Note>

## Customizing the Button

You can modify the button's appearance directly in the style string to match your site's design. Copy the snippet first, then adjust these values:

- **Color** — change `background:#111` to any CSS color value. For example, blue: `background:#3b82f6`
- **Size** — adjust `font-size:22px` to make the emoji icon larger or smaller
- **Position** — change the `bottom:20px` and `right:20px` values to reposition the button along any edge of the screen

<Tip>
  For best results, paste the snippet just before the closing `</body>` tag so your page content loads first.
</Tip>

<Note>
  The Beacons widget runs in an isolated iframe. It does not share data with your website's JavaScript environment.
</Note>
