Plugins extend Overmind's capabilities by adding new tools and behaviors to your conversations. Each plugin is independent — you can toggle them on or off at any time without restarting or losing your chat history.

## Opening the Plugins Panel

Click **Settings** in the sidebar, then select the **Plugins** tab. You will see a list of all available plugins, each with a toggle switch on the right.

---

## Available Plugins

| Plugin | What it does |
|--------|-------------|
| **Response Enhancer** | Applies your selected [Tone setting](/customization/behavior#tone) — Friendly, Formal, Quirky, or Concise — as an active instruction in the system prompt. Enable this to make tone adjustments actually stick in responses rather than being treated as a soft preference. |
| **Smart Calculator** | Gives Overmind access to an accurate math tool. When you ask for calculations, percentages, or unit conversions, Overmind uses this tool instead of approximating in natural language. Also adds a quick-access calculator widget to the input area. |
| **Today's Info** | Provides Overmind with the current date and time. Enable this when you need time-aware responses — for example, "What day is it?" or anything involving scheduling, deadlines, or relative dates. |
| **Business / School Mode** | Forces a strictly professional, calm, and focused tone. Disables emojis, slang, exclamation points, and all casual language. Profanity is never used while this mode is active. Ideal for academic writing, workplace tasks, or any context where formality is required. |

---

## Enabling a Plugin

Click the toggle next to a plugin name. The toggle turns green to confirm the plugin is active. Changes take effect immediately in the current conversation — you do not need to start a new chat.

---

## Disabling a Plugin

Click the toggle again to turn the plugin off. The plugin stops contributing to new messages right away. Existing messages in the conversation are not affected.

---

<Tip>
  Enable **Smart Calculator** any time you need Overmind to handle math — it uses actual computation rather than language-model estimation, giving you accurate results.
</Tip>

<Note>
  **Business / School Mode** overrides the Tone setting while active. Disable it to restore your normal tone preferences.
</Note>
